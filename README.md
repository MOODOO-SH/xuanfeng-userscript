XuanFengEx
==========

QQ旋风Web版增强脚本


基于 @maplebeats 的[js_for_xf](https://github.com/maplebeats/js_for_xf)


Features
--------

- [x] UI精简
- [x] Aria2命令/文件导出
- [x] IDM文件导出
- [x] 自动登录
- [x] magnet链接支持
- [ ] Aria2 RPC
- [ ] bt任务文件夹折叠
